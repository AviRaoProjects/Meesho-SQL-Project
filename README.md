# Meesho-Sales-Analysis
SQL project to analyze e-commerce sales data and extract business insights using aggregation, KPIs, and trend analysis.

## Meesho Sales Analysis (SQL Project)  

**Overview**  
This project analyzes e-commerce sales data using SQL to extract meaningful business insights.  
By performing data validation, KPI analysis, and business-focused queries, the project identifies key factors influencing sales performance—such as product demand, regional distribution, discount impact, and delivery efficiency.  
The goal is to transform raw transactional data into actionable insights for better decision-making.


**Objectives**  
- Analyze overall sales performance using SQL  
- Identify top-performing products and categories  
- Understand regional sales distribution  
- Evaluate impact of discounts on revenue  
- Analyze delivery performance and customer ratings  


**Dataset Information**  
**Rows:** 200  
**Columns:** 12  
**Key Columns:** Order ID, Date, Product, Category, State, Quantity, Price, Discount, Rating, Total Sale  


**Data Categories**  
**Order Details:** Order ID, Order Date  
**Product Information:** Product Name, Category  
**Location Information:** City, State  
**Sales Metrics:** Quantity, Unit Price, Discount, Total Sale  
**Customer Experience:** Delivery Status, Rating  


**Technologies Used**  
- MySQL  
- SQL (Aggregation, Group By, Filtering)  


**Key Insights**  
- A small group of products contributes a significant portion of total revenue  
- Fashion category dominates overall sales performance  
- Top states like Gujarat and Rajasthan contribute majority of revenue  
- Discounted orders increase volume but reduce average order value  
- Strong delivery performance results in high customer satisfaction  


**Conclusion**  
E-commerce sales performance is influenced by product demand, regional distribution, and discount strategies.  
SQL-based analysis helps:  
- Identify high-performing products and regions  
- Understand customer purchasing patterns  
- Evaluate pricing and discount strategies  
- Improve operational efficiency  


**How to Use**  
1. Import the dataset into MySQL
2. Run meesho_sales_final.sql
2.Execute queries to view KPIs and insights
